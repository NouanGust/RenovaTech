# RenovaTech

Repositório central do programa **RenovaTech** — uma iniciativa de capacitação em desenvolvimento de jogos e programação, realizada em parceria com a **GameCreators**.

Aqui você encontra os projetos práticos desenvolvidos ao longo do programa, cobrindo desde fundamentos de lógica e web até desenvolvimento de jogos com Unity.

---

## Projetos

### Sunny Runner — Endless Runner

> Jogo 2D de corrida infinita desenvolvido em Unity com geração procedural de obstáculos, parallax, sistema de pontuação e gerenciamento de cenas.

Projeto prático que abrange os fundamentos de game development com Unity: física 2D, prefabs, corrotinas, padrão Singleton, UI com TextMeshPro, áudio e persistência de dados com PlayerPrefs.

**Tecnologias:** Unity · C# · URP · Unity Input System · Tilemaps · Physics 2D · TextMeshPro

### CineLib — Biblioteca de Filmes

> Site e API REST de biblioteca de filmes desenvolvidos com Python e Flask. Permite cadastrar, listar, editar e remover filmes através de uma interface web e de endpoints de API REST.

Projeto full-stack com foco nos fundamentos de desenvolvimento back-end com Python: rotas com Flask, operações CRUD, persistência de dados com SQLite, templates com Jinja2 e integração entre front-end e back-end.

**Tecnologias:** Python · Flask · SQLite · Jinja2 · HTML5 · CSS3 · REST API

| | |
|---|---|
| Repositório | [github.com/NouanGust/Lista_de_filmes_python_flask](https://github.com/NouanGust/Lista_de_filmes_python_flask) |

### Palavra Secreta — Jogo de Adivinhação

> Jogo de palavras em HTML, CSS e JavaScript puro. O jogador recebe dicas progressivas e tem tentativas limitadas para descobrir a palavra secreta — a cada erro, uma nova dica é revelada.

Projeto web sem dependências externas, focado nos fundamentos de front-end: estrutura semântica, Flexbox, manipulação do DOM, arrays de objetos, lógica condicional e geração de números aleatórios.

**Tecnologias:** HTML5 · CSS3 · JavaScript (ES6+)

### TudoSobreGatos

> Aplicação web desenvolvida durante as aulas do RenovaTech para consumir APIs públicas e exibir fatos e imagens aleatórias sobre gatos em tempo real.

Projeto front-end focado em consumo de APIs com JavaScript, cobrindo requisições assíncronas, manipulação do DOM, atualização dinâmica da interface e filtragem de resultados com base em metadados das imagens.

**Tecnologias:** HTML5 · CSS3 · JavaScript (ES6+) · fetch · async/await · APIs públicas

**APIs utilizadas:** Cat Fact Ninja · The Cat API

|---|---|
| Versão publicada | [github pages](https://nouangust.github.io/JavaScriptAPIAula/) |

### FlappyBox — Flappy Bird Style

> Jogo 2D estilo Flappy Bird desenvolvido no Construct 3. O jogador controla uma caixa que deve desviar de obstáculos enquanto avança infinitamente.

Projeto de game development focado em game design com Construct 3: física 2D, colisão, geração procedural de obstáculos, sistema de pontuação, gerenciamento de estados (menu, jogo, game over) e publicação para web.

**Tecnologias:** Construct 3 · JavaScript · HTML5 · Canvas 2D

## Sobre o Programa

O **RenovaTech** foi um programa de capacitação em tecnologia e desenvolvimento de jogos voltado para a formação prática de novos desenvolvedores. Os projetos aqui listados foram construídos durante as aulas e refletem os conteúdos abordados em cada etapa do curso.
